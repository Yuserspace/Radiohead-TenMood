Radiohead TenMood
​라디오헤드(Radiohead) 정규 1집부터 9집까지 총 100곡의 수록곡을 10가지 감정 카테고리로 큐레이션한 아카이브 웹페이지입니다.
​주요 기능
​10가지 무드 필터링: 우울, 분노, 불안, 좌절, 공허, 기쁨, 반전, 서늘, 따뜻, 신남 감정에 따른 곡 추천.
​앨범별 아카이빙: 정규 1집부터 9집까지 앨범별로 곡 확인 가능.
​다중 필터: 무드와 앨범을 중복 선택하여 원하는 조건의 곡을 교차 검색.
​음원 바로가기: 각 곡의 공식 유튜브 뮤직 검색 결과로 바로 연결.
​사용 방법
​상단의 버튼을 클릭하여 원하는 감정(무드)과 앨범을 선택합니다.
​카드 상단의 앨범 정보와 감정 태그를 확인합니다.
​곡 하단의 유튜브 뮤직 아이콘을 클릭하면 해당 곡의 검색 페이지로 이동합니다.
​기술 스택
​HTML5, CSS3, JavaScript (Vanilla)
​외부 API 연동 없이 데이터 로컬 관리
